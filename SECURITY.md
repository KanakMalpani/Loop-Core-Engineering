# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |

## Reporting a vulnerability

Please report security issues **privately** via [GitHub Security Advisories](https://github.com/KanakMalpani/Loop-Core-Engineering/security/advisories/new) or email the repository owner.

Do not open public issues for undisclosed vulnerabilities.

## Scope

This repo contains specifications and validation tooling only — no network services or production runtime.
