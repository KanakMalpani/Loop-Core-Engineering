# RFCs — Loop Engineering core specifications

Normative changes to LSS, LES, or `loop-ids.md` start here until `12-loop-institute` owns governance.

## Process

1. Copy [`templates/rfc-template.md`](../templates/rfc-template.md) to `rfcs/NNNN-short-title.md`.
2. Open PR with label `rfc`.
3. Require: problem, proposed change, migration, backwards compatibility.
4. On accept: bump semver in `CHANGELOG.md` and implement schema/doc changes.
